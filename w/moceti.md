# moceti


1. to deliver, set free, release, cause one’s release or deliverance from (abl.). imper praes. *mocehi* Pv.ii.1#6 (duggatiyā); Pv\-a.12; aor. *[mocesi](mocesi.md)* Pv\-a.112 (dāsavyato); ger. *[mocetvā](mocetvā.md)* Pv\-a.8, Pv\-a.77; inf mocetuṃ Pv\-a.45 (petalokato).
2. to discharge, emit (semen in coitu) Vin.iii.36, Vin.iii.39 (as Caus II.), Vin.iii.110.
3. to let loose, set into motion, stir: *padaṃ* m. to run Ja.iii.33
4. to discharge, fulfil: *paṭiññaṃ* one’s promise Dhp\-a.i.93.
5. to unharness Dhp\-a.i.67.
6. to detach SN.i.44

caus 2 *[mocāpeti](mocāpeti.md)* to cause to be freed, to give freedom, to let loose Vin.iv.316 (opp. bandhāpeti).

Caus. of muñcati
